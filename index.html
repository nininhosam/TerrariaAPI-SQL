rotas:

Rota GET => Traz uma lista de todos os itens registrados no banco de dados

Rota GET/${item} => Aceita um item ${item} registrado e traz apenas a sua arvore de crafting

Rota POST/ => Adiciona ao banco de dados apartir de uma JSON com 3 paramêtros: "name", "obtained" e "recipe"(opcional)
"name" deverá ser uma string com o nome do objeto
"obtained" deverá ser uma string com o método de obtenção
"recipe" deverá ser um objeto onde suas chaves são os materiais, e seus valores a quantidade.

Rota PUT/${item} => Aceita uma JSON assim como o post, e corrige o item ${item}

Rota DELETE/${item} => Deleta um item ${item}
